BlogSpace 🚀
Welcome to BlogSpace, a clean and modern blogging platform template designed to share your thoughts and stories. This project is built purely with HTML and CSS, focusing on a beautiful user interface, smooth animations, and a responsive layout.

✨ Features
Purely Frontend: Built with only HTML and CSS, making it lightweight and easy to understand.

Stunning UI: A dark-themed, visually appealing design with a starry night hero section.

Smooth Animations: Subtle animations like twinkling stars and fade-in effects to enhance the user experience.

Fully Responsive: The layout adapts seamlessly to different screen sizes, from mobile devices to desktops.

Easy Navigation: A fixed header with smooth scrolling links to different sections of the page.

Clean Code: Well-structured and commented code for easy customization.
